# Plant Society
This is a demo project to build a blog using python, mysql, html, css and javascript.

## Dependencies
* python v3.7.1
* pip 18.0

## Setup
```bash
git clone https://github.com/anusreesubash/plant-society.git
```

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python app.py
```
